Projects

Fuzzer [Python]
October 2014

A web crawling fuzz testing tool designed to find weaknesses in a program's attack surface. The fuzzer checks for common exploits to vulnerability such as SQL injects, cross-site scripting, buffer overflows, and guessing input and authorization using a provided file of common words.

Medical Imaging Program [Java]
April 2014

A simple medical imaging program with capabilities to perform basic image processes such as adjusting image contrast and brightness, cropping, and resizing. Basic file and directory manipulation such as copying was also included. The focus of this project was to get familiar with common design patterns and apply them to a real world application through architecture design and implementation.

Tool Share [Python, Django Framework, HTML, CSS]
December 2013

A web application for people to share tools within their community. In the planning phase, I acted as the document coordinator by making sure everything was documented and ensuring consistency throughout. During development, I worked on some implementation as well as quality assurance by testing features and debugging.
